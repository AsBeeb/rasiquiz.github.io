Creation of the title:

titleSetup(mode,title,id);

Functions for animating in and out:

titleAnimIn(id);
titleAnimOut(id);

Animation example:



Modes:

'write' is used when the function is invoked in a <script> tag.

'return' is used when the function is used in javascript.


Title:

The title is simply the title you want to be shown.

Example: 'random title'


ID:

The id is a unique string used when invoking the animations.

Example: 'uniqueid'


Example:

Here the 'write' mode is used and the title being set is 'Text' and the title is given the unique id 'text1'

<script>titleSetup('write','Text','text1')</script>

Would output:

onclick="titleAnimIn('text1')"
onclick="titleAnimOut('text1')"
